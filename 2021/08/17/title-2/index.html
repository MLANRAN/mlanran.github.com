博客的搭建

Share

一、配置GitHub
在GitHub注册一个账号（有的登录就可以）
记住username 后面要用到
创建一个repository储藏库；
名字填写刚记住的用户名+<.github.io>即可。

二、安装git、node
网上找，国内镜像源也可以。
都选择默认属性安装既可以
相关下载地址：（非实时，有时效性，会过期）
node：https://link.zhihu.com/?target=https%3A//nodejs.org/en/
git：https://link.zhihu.com/?target=https%3A//github.com/waylau/git-for-win

安装完git后进行设置：
打开git bash 设置名称和邮箱为GitHub上设置的（我的是MLANRAN）；
相关代码：

git config –global user.name “MLANRAN”

git config –global user.email “964308386@qq.com“

接下载安装hexo 可以利用cmd安装

npm install -g hexo-cli
安装hexo 相关命令的东西

npm install hexo-deployer-git –save

cmd有 好用的第三方软件cmder可以百度上官网安装。(注意添加环境变量）如果文件夹右键没有启动选项的话，打开cmd运行下列命令。即可

Cmder.exe /REGISTER ALL

三、设置
新建项目的文件夹，此处用test；
键盘按住shift鼠标右键，选择cmder here 没有用cmd也是一样的；

hexo init blog
注意：一定要是运行的环境是当前测试的文件夹
成功后会有下列提示：

下一步是启动本地服务器
运行前cd当前环境到刚刚用hexo产生的文件夹下即

F:\test\blog
才可以运行下列操作()

hexo generate
hexo server

此时静态的本地网页已经做好了；（根据cmder提示网址为：

localhost.com

下一步是把网页发布到GitHub上去；

ssh-keygen -t rsa -C “GitHub注册的邮箱地址”
会询问一系列问题，包括保存位置，pass phrase等等都回车跳过即可。
会出现下列信息表示成功

找到文件并且打开复制里边的信息，ssh-rsa 开头的sshkey
登录GitHub并且进入到设置里边 找到SSH and GPG Keys设置选项

添加到博客中即可。

四、配置博客

在创建的blog文件目录下，打开_config.yml文件，修改参数信息

Source: https://github.com/hexojs/hexo/

创建新文章：

hexo new “<文件名>”
#INFO created: <文件地址（一般是创建test下的source↓_posts下的文件>

修改文件为你想要的内容保存即可；

保存完成后，依次进行下列的操作
1、

hexo clean

2、

hexo generate

3、

hexo server

4、这步执行完成后可以在网上看到自己的杰作（有延迟，是GitHub上的延时）过一段时间后刷新即可
文章发布到网上

hexo deploy
